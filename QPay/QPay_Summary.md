# QPay - Comprehensive Service Overview

## 1. Overview of QPay Service

QPay is a user-friendly, secure, and dependable payment solution for everyday usage developed by Innovation Investment LLC in Mongolia. Key highlights include:

- **Launch Year**: Development started in 2016, company established in 2017
- **Market Position**: Holds 4.1% of Mongolia's non-cash payment market share as of 2022
- **User Base**: 2.8 million consumers
- **Transaction Volume**: 63 million transactions in 2022
- **Merchant Integration**: 7,300+ entities integrated within the system
- **Bank Partnerships**: Integrated with all 11 commercial banks in Mongolia

## 2. Main Features and Capabilities

### Core Payment Services
- **Digital Wallet**: Mobile payment application supporting multiple payment methods
- **QR Code Payments**: Scan-to-pay functionality for goods and services
- **P2P Transfers**: Peer-to-peer payment transfers between QPay accounts
- **Card Tokenization**: Secure tokenization of bank cards from any Mongolian bank
- **Multi-Bank Support**: Users can choose from all 11 commercial banks in Mongolia

### QR Code Implementation
- **Merchant Mode**: QR displayed on merchant screen or counter sticker

- **Dynamic QR**: Generated for specific transactions
- **Static QR**: Permanent QR for merchant locations

## 5. Payment Flow and Process

### Card Tokenization Process
1. User inputs 16-digit card number, expiry date, and CVV code
2. Internet PIN code verification
3. 3DES standard authentication for international cards
4. Card tokenized and stored securely in QPay wallet
5. Future transactions use token instead of card details

### Transaction Flow
1. **Initiation**: Customer scans QR code or enters payment details
2. **Authentication**: User authenticates via PIN or biometric
3. **Processing**: Transaction routed through appropriate bank
4. **Settlement**: Real-time settlement between banks
5. **Confirmation**: Instant notification to both parties

### Commission Structure
- **P2P Transactions**: 100 MNT fee between bank clients
- **Government Services**: Standard transaction fees apply
- **Card Transactions**: 1% commission split between issuing and acquiring banks
