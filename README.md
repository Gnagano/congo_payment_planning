## 見積もり履歴
- **2025-06-03版**: 139人月 / 55,600,000円（フル機能版）
- **2025-06-19版**: 75人月 / 30,000,000円（MVP版）

## 主要機能
- QR決済システムコア機能
- 銀行連携システム
- ウォレット機能
- 監視・運用システム

## 開発戦略
MVP（最小実行可能製品）アプローチを採用し、Static QR決済から開始して段階的に機能拡張を行う計画です。
