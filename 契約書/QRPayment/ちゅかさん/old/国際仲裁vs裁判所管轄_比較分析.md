# 国際仲裁 vs 裁判所管轄 比較分析

作成日: 2025-07-21  
対象契約: GOODTECH SOFT LLC（モンゴル）との開発委託契約

## 背景情報

- **当事者**: Curiox（香港）とGOODTECH（モンゴル）
- **契約金額**: 28,800,000円
- **共通言語**: 日本語
- **問題点**: 日本・香港とモンゴル間に相互執行条約がない

## 1. 日本裁判所管轄

### メリット
- **言語面での利便性**
  - 日本語での訴訟手続き
  - 通訳・翻訳コストの削減
- **法的安定性**
  - 確立された判例・実務
  - 予測可能性が高い
- **コスト面**
  - 弁護士費用が比較的安価
  - 手続き費用も明確

### デメリット
- **執行の困難性**
  - モンゴルでの判決執行には別途手続きが必要
  - 執行までに1-2年かかる可能性
  - 執行が事実上不可能なケースも
- **実効性の欠如**
  - 勝訴しても回収できないリスク
  - 二重の費用負担（日本での訴訟＋モンゴルでの執行）

## 2. 国際仲裁

### メリット
- **執行の容易性**
  - ニューヨーク条約（1958年）により160カ国以上で執行可能
  - モンゴルも加盟国（1994年加盟）
  - 執行拒否事由が限定的
- **手続きの迅速性**
  - 通常6-12ヶ月で仲裁判断
  - 一審制で終結
- **中立性**
  - どちらの国にも偏らない
  - 仲裁人を当事者が選定可能
- **秘密性**
  - 非公開手続き
  - ビジネス情報の保護

### デメリット
- **高額な費用**
  - 仲裁機関への申立費用
  - 仲裁人報酬（1人あたり数百万円）
  - 総額で500-1000万円程度
- **上訴不可**
  - 仲裁判断は最終的
  - 誤った判断でも覆せない
- **手続きの複雑性**
  - 仲裁規則の理解が必要
  - 仲裁人選定プロセス

## 3. 費用比較

### 日本裁判所
- **訴訟費用**: 100-200万円
- **弁護士費用**: 200-500万円
- **モンゴルでの執行費用**: 200-300万円
- **合計**: 500-1000万円

### 国際仲裁（JCAA）
- **管理料金**: 50-150万円
- **仲裁人報酬**: 200-400万円（1名）
- **弁護士費用**: 200-400万円
- **合計**: 450-950万円

## 4. 推奨される仲裁機関

### JCAA（日本商事仲裁協会）
- **メリット**: 日本語対応、費用が比較的安価
- **デメリット**: 国際的知名度が低い

### SIAC（シンガポール国際仲裁センター）
- **メリット**: アジアで最も評価が高い、執行実績豊富
- **デメリット**: 英語手続き、費用が高額

### ICC（国際商業会議所）
- **メリット**: 世界的に最も権威がある
- **デメリット**: 最も高額、手続きが厳格

## 5. 今回の契約への推奨

### 推奨案: JCAA仲裁条項

**理由**:
1. **執行可能性**: モンゴルでの執行が現実的
2. **費用対効果**: 契約金額（2,880万円）に対して妥当
3. **言語**: 日本語での手続きが可能
4. **立地**: 東京での仲裁は両当事者にとって中立

### 契約条項案

```
第○条（紛争解決）
1. 本契約から生じる紛争については、まず当事者間で誠実に協議し、
   協議開始から30日以内の解決を図るものとする。

2. 協議により解決できない場合は、一般社団法人日本商事仲裁協会の
   商事仲裁規則に従い、東京において仲裁により最終的に解決する。

3. 仲裁人は1名とし、仲裁言語は日本語とする。

4. 本契約の準拠法は日本法とする。
```

## 6. 結論

国際取引において執行可能性は極めて重要です。特に相互執行条約がない国との取引では、**国際仲裁条項が必須**と考えられます。

初期費用は高額ですが、実際に紛争が生じた際の**回収可能性**を考慮すると、国際仲裁の選択が最も合理的です。
