# 契約書変更比較
## 2025年7月18日 → 2025年7月23日

本文書は、2025年7月18日版と2025年7月23日版のソフトウェア開発委託契約書の変更点を詳細に比較したものです。

---

## 変更された条項

### 第８条（原始資料等の保管管理）

**変更前（2025-07-18）:**
```
第８条　乙は原始資料等を、善良な管理者の注意義務をもって管理、保管し、本件業務の遂行以外の目的には、一切使用しないものとする。
```

**変更後（2025-07-23）:**
```
第８条　乙は原始資料等を、善良な管理者の注意義務をもって管理、保管し、本件業務の遂行以外の目的には、一切使用しないものとする。乙は本契約終了後1年間、開発環境、技術資料、ソースコードのバックアップを保管する
```

**変更内容:**
- 契約終了後のバックアップ保管義務が追加された
- 保管期間：1年間
- 保管対象：開発環境、技術資料、ソースコードのバックアップ

### 第１５条（検品）

**変更前（2025-07-18）:**
```
1. 甲は、乙より成果物の納入がなされた日から10日以内（以下「検査期間」という）に、成果物の検査を行い、その検査結果について乙に通知するものとする。但し、過誤その他の瑕疵があったときは、直ちに乙に通知するものとする。
2. 乙による成果物納品の日から10日経過した場合は、前項の検査結果の通知にかかわらず、当該成果物は前項所定の検査に合格したものとみなす。甲が正当な理由なく成果物の受領を拒否し、乙が甲へ当該成果物を納入した日から前項の期間を経過したときも同様とする。
```

**変更後（2025-07-23）:**
```
1. 甲は、乙より成果物の納入がなされた日から20営業日以内（以下「検査期間」という）に、成果物の検査を行い、その検査結果について乙に通知するものとする。但し、過誤その他の瑕疵があったときは、直ちに乙に通知するものとする。
2. 乙による成果物納品の日から20営業日経過した場合は、前項の検査結果の通知にかかわらず、当該成果物は前項所定の検査に合格したものとみなす。甲が正当な理由なく成果物の受領を拒否し、乙が甲へ当該成果物を納入した日から前項の期間を経過したときも同様とする。
```

**変更内容:**
- 検査期間が「10日」から「20営業日」に延長された
- 合格みなし期間も同様に「10日」から「20営業日」に延長された

### 第１８条（著作権の帰属）

**変更前（2025-07-18）:**
```
第１８条　納入物のうち本件プログラムの著作物について、本件プログラムに結合され又は組み込まれたもので、乙が従前から有していたプログラム（コンテンツ及びデータベースを含む。以下同じ）及び乙が本件業務の実施中新たに作成したプログラムの著作権は、乙に留保されるものとする。但し、甲は、納入された本件プログラムの著作物の複製物を、かかる成果物を自己使用の範囲内に限って自由に使用し、又は著作権法規定に基づき複製、翻案することができる。
```

**変更後（2025-07-23）:**
```
第１８条　新規開発部分の著作権は甲乙の共同所有とし、各自が自由に利用、改変、第三者への利用許諾（エンドユーザーへの譲渡を含む）を行えるものとする。
```

**変更内容:**
- 著作権の帰属が大幅に変更された
- 変更前：乙（受託者）に著作権が留保され、甲は自己使用の範囲内でのみ利用可能
- 変更後：甲乙の共同所有となり、両者が自由に利用・改変・第三者への利用許諾が可能

## 新規追加された条項

### 第２４条（準拠法）

**新規追加内容:**
```
第２４条　本契約は、日本国の法律を準拠とする。
```

### 第２５条（進捗報告）

**新規追加内容:**
```
第２５条　システム開発期間において2週間毎に定例進捗報告を行い、その形態は会議あるいはレポート形式で行えるものとする。
```

### 第２６条（優先交渉権）

**新規追加内容:**
```
第２６条　本契約の成果物に関する追加開発、機能拡張、保守業務について、甲は乙に対して優先的に交渉する機会を設ける
```

### 第２７条（開発体制の継続性）

**新規追加内容:**
```
第２７条　今後の追加開発においては、可能な限り本契約と同じ開発チームによる対応を考慮する
```

## 削除された条項

削除された条項はありません。

## 変更サマリー

1. **契約終了後の対応強化**
   - バックアップ保管義務の追加（第８条）

2. **検査期間の延長**
   - 10日から20営業日への延長（第１５条）

3. **著作権の共有化**
   - 乙単独所有から甲乙共同所有への変更（第１８条）
   - 両者の自由な利用・改変・第三者への利用許諾が可能に

4. **法的基盤の明確化**
   - 準拠法条項の追加（第２４条）

5. **プロジェクト管理の強化**
   - 定期進捗報告の義務化（第２５条）

6. **継続的な関係性の確保**
   - 優先交渉権の付与（第２６条）
   - 開発チームの継続性配慮（第２７条）

これらの変更により、契約はより公平で、両者の長期的な協力関係を重視した内容となっています。
