# 契約書変更比較レポート
## 2025-07-23版 → 2025-07-27版 → 2025-07-28版

比較対象：ソフトウェア開発委託契約書（GOODTECH SOFT LLC）

---

## 重要な変更点

### 1. 第４条（成果物の納入）- 成果物の詳細定義追加

#### 変更前（2025-07-23版）
```
第４条　乙は、甲が指定する期限までに、成果物を納入する。但し、次の各号に該当する場合には、乙は、甲に対し成果物の納入期限の変更を求めることができるものとする。
```

#### 変更後（2025-07-27版）
```
第４条
1. 乙は、甲が指定する期限までに、成果物を納入する。成果物として機能毎に次のように定める。

| 項番 | 機能分類 | 成果物 |
|------|----------|---------|
| （あ） | Static QRトランザクション | 送金テスト結果 |
| （い） | ウォレット | アプリテスト結果 |
| （う） | クレーム処理、トランザクション修正システム | クレーム対応シナリオテスト結果 |
| （え） | Dev-Opsモニタリングシステム | 開発導入一連の作業マニュアルと動作確認 |
| （お） | サービス全体モニタリング | モニタリングと動作マニュアルと勉強開催 |

2. 次の各号に該当する場合には、乙は、甲に対し成果物の納入期限の変更を求めることができるものとする。
```

**変更の影響：**
- 各機能の成果物が具体的に定義され、納品物が明確化
- テスト結果、マニュアル、動作確認などの具体的な成果物が明記
- 契約履行の判定基準がより明確に

---

### 2. 第１８条（著作権の帰属）- 既存機能の著作権に関する規定追加

#### 変更前（2025-07-23版）
```
第１８条　新規開発部分の著作権は甲乙の共同所有とし、各自が自由に利用、改変、第三者への利用許諾（エンドユーザーへの譲渡を含む）を行えるものとする。
```

#### 変更後（2025-07-27版）
```
第１８条　新規開発部分の著作権は甲乙の共同所有とし、各自が自由に利用、改変、第三者への利用許諾（エンドユーザーへの譲渡を含む）を行えるものとする。今回開発機能は全て新規機能であり、今後追加で開発続行する場合、今回開発した機能は既存機能となる。既存機能の著作権に関して、所有者へそのまま属する。
```

**変更の影響：**
- 今回の開発が全て新規機能であることを明確化
- 将来の追加開発時における著作権の取り扱いを明記
- 既存機能の著作権は現在の所有者に帰属することを確認

---

## 変更による契約上の留意点

1. **成果物の明確化**により、納品時の検収基準が具体的になり、双方の認識の相違を防ぐことが可能
2. **著作権条項の追加**により、将来の追加開発時の権利関係が明確化され、紛争リスクが軽減

---

## 追加変更点（2025-07-27版 → 2025-07-28版）

### 3. 第４条（成果物の納入）- 勉強開催の詳細化

#### 変更前（2025-07-27版）
```
| （お） | サービス全体モニタリング | モニタリングと動作マニュアルと勉強開催 |
```

#### 変更後（2025-07-28版）
```
| （お） | サービス全体モニタリング | モニタリングと動作マニュアルと勉強開催（３回まで、１回は2時間） |
```

**変更の影響：**
- 「勉強開催」の内容が具体的に定義され、契約履行の基準が明確化
- 回数制限（3回まで）と時間制限（1回2時間）により、乙の負担が限定的に
- 成果物の検収時に具体的な判定基準として機能

---

### 4. 第２４条（準拠法）- 裁判管轄の追加

#### 変更前（2025-07-27版）
```
第２４条　本契約は、日本国の法律を準拠とする。
```

#### 変更後（2025-07-28版）
```
第２４条　本契約は、日本国の法律を準拠とし、裁判管轄は東京都とする。
```

**変更の影響：**
- 国際契約における最重要課題の解決
- 紛争発生時の裁判管轄が明確になり、モンゴルでの訴訟リスクを回避
- 法的安定性の大幅な向上と紛争解決コストの予測可能性

---

## 総合的な改善評価

### 段階的改善の推移
1. **2025-07-23版**: 基本的な契約条件の確立
2. **2025-07-27版**: 成果物の明確化と著作権の詳細化
3. **2025-07-28版**: 実務的詳細の具体化と紛争解決メカニズムの完成

### 契約リスクの軽減
- **成果物リスク**: 具体的定義により大幅軽減
- **紛争解決リスク**: 東京都管轄により解決
- **著作権リスク**: 詳細な規定により明確化

2025-07-28版により、実務上の課題がほぼ解決され、国際ソフトウェア開発契約として十分な品質に到達しました。
