## 1. 主要な追加・修正項目

### 🆕 **新規追加された条項**

#### **1.1 第17条（保守サービス）- 大幅拡充**

**元版**:
```
第17条　甲及び乙は、次の各号に掲げる保守等に関する契約を別途締結できるものとする。
1. 保証期間経過後の本ソフトウェアの不稼働を含む稼働不良に対する技術サービス
2. 保証期間経過後の本ソフトウェア、成果物の瑕疵に対する修補
3. 乙の責に帰すべからざる事由による本件ソフトウェアの不稼働を含む稼働不良に対する技術サービス
```

**修正版**:
```
第17条　乙は、成果物の納品後、以下のサポートを無償で提供する。
1. 納品後6ヶ月間：月間20時間までの技術サポート
2. 納品後12ヶ月間：重大バグ（システム停止・機能不全）の無償修正
3. 緊急時（営業時間外）の連絡体制確保
4. 保証期間経過後の本ソフトウェアの不稼働を含む稼働不良に対する技術サービス（別途契約）
5. 保証期間経過後の本ソフトウェア、成果物の瑕疵に対する修補（別途契約）
```

**変更効果**: 別途契約 → **無償サポート6ヶ月・重大バグ12ヶ月対応**

#### **1.2 第19条（責任・補償条項）- 完全新規**

**元版**: なし

**修正版**:
```
第19条　乙は、以下の責任を負う。
1. 第三者の知的財産権侵害に対する補償（委託料の50％を上限とする）
2. 成果物の瑕疵による損害賠償（委託料の100％を上限とする）
3. 甲が第三者に負担した損害について、乙の責に帰すべき事由がある場合の按分負担
```

**変更効果**: 責任条項なし → **明確な責任・補償体制確立**

#### **1.3 第27条（上位契約との連携）- 完全新規**

**元版**: なし

**修正版**:
```
第27条　乙は、甲の上位契約（エンドクライアント）の納期管理に協力する義務を負う。
1. 乙は、遅延リスクを認識した場合、甲に対して速やかに報告し、甲がエンドクライアントとの調整を行えるよう協力する。
2. 甲がエンドクライアントから遅延了承を得た場合、乙の納期も同様に調整される。
3. 乙は、甲の上位契約における遅延ペナルティ回避のため、以下の協力を行う：
   - 遅延予想時の15日前事前通知
   - 遅延原因と対策案の詳細報告
   - 甲の要請に応じた進捗状況の詳細報告
```

**変更効果**: 連携条項なし → **上位契約との一体的管理体制**

### 🔄 **大幅に修正された条項**

#### **1.4 第4条（成果物の納入）- 納期調整**

**元版**:
```
2. 次の各号に該当する場合には、乙は、甲に対し成果物の納入期限の変更を求めることができるものとする。

   2. 甲により、本件業務の内容が変更されたとき。
   3. 天災事変その他の不可抗力によって、納入期限までに成果物を納入することが困難になったとき。

**修正版**:
```
2. 最終納期は2025年12月1日とし、別途調整により変更される場合がある。次の各号に該当する場合には、乙は、甲に対し成果物の納入期限の変更を求めることができるものとする。
   1. 天災事変その他の不可抗力によって、納入期限までに成果物を納入することが困難になったとき。
   2. 甲により、本件業務の内容が重大に変更されたとき。
   3. 原始資料その他本件業務遂行に必要な資料、情報、機器等の提供の懈怠、遅延等のため本件業務の進捗に支障が生じたときであって、その支障の直接的原因又は間接的原因が、乙に起因するものではないとき。
```

**変更効果**: 変更事由の順序変更・明確化、**別途調整による柔軟性確保**

#### **1.5 第15条（検品）- 検査期間短縮**

**元版**:
```
1. 甲は、乙より成果物の納入がなされた日から20営業日以内（以下「検査期間」という）に、成果物の検査を行い、その検査結果について乙に通知するものとする。
2. 乙による成果物納品の日から20営業日経過した場合は、前項の検査結果の通知にかかわらず、当該成果物は前項所定の検査に合格したものとみなす。
```

**修正版**:
```
1. 甲は、乙より成果物の納入がなされた日から15営業日以内（以下「検査期間」という）に、成果物の検査を行い、その検査結果について乙に通知するものとする。
2. 乙による成果物納品の日から15営業日経過した場合は、前項の検査結果の通知にかかわらず、当該成果物は前項所定の検査に合格したものとみなす。
```

**変更効果**: 20営業日 → **15営業日（検査期間短縮）**

#### **1.6 第18条（著作権の帰属）- 所有権統一**

**元版**:
```
第18条　新規開発部分の著作権は甲乙の共同所有とし、各自が自由に利用、改変、第三者への利用許諾（エンドユーザーへの譲渡を含む）を行えるものとする。

**変更効果**: 共同所有 → **甲（Curiox）の単独所有**
