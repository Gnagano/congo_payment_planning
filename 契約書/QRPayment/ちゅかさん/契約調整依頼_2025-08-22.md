# 契約調整依頼書

### **主要な調整項目**

1. 著作権・知的財産権の変更（共同所有 → 単独所有）
2. 責任・補償条項の新規追加
3. 上位契約連携条項の新規追加

---

## 1. 著作権・知的財産権の調整

### **現在の条項（第 18 条）**

```
新規開発部分の著作権は甲乙の共同所有とし、各自が自由に利用、改変、
第三者への利用許諾（エンドユーザーへの譲渡を含む）を行えるものとする。
```

### **調整提案：共同所有から単独所有への変更**

**重要**: 上位契約（エンドユーザー）との整合性確保のため、**共同所有から単独所有に変更**します。

#### **変更後の条項**

```
新規開発部分の著作権は甲の単独所有とし、甲が自由に利用、改変、
第三者への利用許諾（エンドユーザーへの譲渡を含む）を行えるものとする。
```

### **変更理由**

1. **エンドユーザー要求**: 金融機関は著作権の単独所有を強く要求
2. **規制要件**: 金融庁監督指針における管理責任の明確化
3. **権利関係簡素化**: 複雑な共同所有関係の回避

---

## 2. 責任・補償条項の新規追加

### **現在の状況**

元の契約書には責任・補償に関する明確な条項がありませんでした。

### **調整提案：責任・補償条項の新規追加**

リスク管理の観点から、以下の責任・補償条項を新規追加したいと考えております：

#### **新規追加条項（第 19 条）**

```
乙は、以下の責任を負う。

1. 第三者の知的財産権侵害に対する補償（委託料の50％を上限とする）
2. 成果物の瑕疵による損害賠償（委託料の100％を上限とする）
3. 甲が第三者に負担した損害について、乙の責に帰すべき事由がある場合の按分負担
```

### **追加理由**

- **リスク管理**: 適切なリスク分散の実現
- **業界標準**: 金融システム開発の標準的な責任設定
- **契約整合性**: 上位契約との責任体系の統一

---

## 3. 上位契約連携条項の追加

### **新規追加提案**

```markdown
第 27 条　上位契約との連携

1. **連携協力義務**
   乙は、甲の上位契約（エンドクライアント）の納期管理に協力する。

2. **遅延リスク管理**
   遅延リスクを認識した場合、甲に対して速やかに報告し、
   甲がエンドクライアントとの調整を行えるよう協力する。

3. **ペナルティ回避協力**
   甲の上位契約における遅延ペナルティ回避のため、以下の協力を行う：

   - 遅延予想時の 15 日前事前通知
   - 遅延原因と対策案の詳細報告
   - 甲の要請に応じた進捗状況の詳細報告

4. **納期調整の連動**
   甲がエンドクライアントから遅延了承を得た場合、
   乙の納期も同様に調整される。
```

### **追加理由**

- **一体的管理**: プロジェクト全体の一体的なリスク管理
- **ペナルティ回避**: 上位契約の遅延ペナルティ回避への協力
- **情報共有**: 適切な情報共有による問題の早期発見・対応
